# Программа установщик/деинсталятор на C#
Программа написана на C# с использованием Windows Form, ключевые функции:
1.Устанавливается в тот путь, который вы выбрали
2.Программа умеет рабоать с системным реестром windows, создает новые пути
3.Использование многопоточности, использует все ядра и потоки процессора, равномерно распределяя нагрузку на этапе установки и удаления
4.Поддержка работы с командной строкой cmd, имеет такие функции как:
/install автоматический режим
/silent полностью автоматический режим, без диалоговых окон
/uninstall удаление программы через командную строку
5.Деинсталятор можно открыть в панеле управления, удаление программ, там находите и удаляете эту программу
6.Во время удаления программы, удаляются созданные ярлыки, изменения в системном реестре, сама папка с программой, куда она была установленна и временные файлы.
